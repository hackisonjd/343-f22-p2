This app is designed with heavy inspiration from a weather station I had growing up, the Bushnell Wireless FX 5. It is designed to show the weather given a location check.

The app connects to two APIs, the Weather Forecast API provided by Open-Meteo, and the Geolocation API present in many browsers natively.

This app satisfies the requirements of the assignment because it is responsive, uses two different APIs, and displays results to a query.
